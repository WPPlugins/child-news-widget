== Description ==

This plugin will display links to posts from a category in a widget that will be assigned to a page, and will display on that page and on all child pages.
== Installation ==

1. Upload `child-news-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
